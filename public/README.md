# Public Assets Folder

This folder contains static assets like images that will be served by Next.js.

## Image Setup Instructions

1. Copy your `Eigen onboarding event.jpg` file to this `public` folder
2. Rename it to: `eigen-onboarding-event.jpg` (all lowercase, with hyphens)
3. The image will be accessible at `/eigen-onboarding-event.jpg` in your website

## File Structure

```
public/
  ├── eigen-onboarding-event.jpg  (image.png)
  └── README.md                    (this file)
```

## Note

- All files in the `public` folder are served statically
- Use lowercase filenames with hyphens for best compatibility
- Supported image formats: .jpg, .jpeg, .png, .webp, .gif
