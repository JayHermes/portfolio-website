# Public Assets Folder

This folder contains static assets like images that will be served by Next.js.

## Image Setup Instructions

### Eigen Event Image
✅ `eigen-onboarding-event.jpg` - Already added

### CodeStory{Uyo} Image
1. Download an image from the Google Drive folder: https://drive.google.com/drive/folders/163GClV-2e_yF9GYPL09ewRi-TXJGaP9D
2. Copy the image file to this `public` folder
3. Rename it to: `codestory-uyo.jpg` (all lowercase, with hyphens)
4. The image will be accessible at `/codestory-uyo.jpg` in your website

## File Structure

```
public/
  ├── eigen-onboarding-event.jpg  ✅ (already added)
  ├── codestory-uyo.jpg           (download from Google Drive)
  └── README.md                    (this file)
```

## Note

- All files in the `public` folder are served statically
- Use lowercase filenames with hyphens for best compatibility
- Supported image formats: .jpg, .jpeg, .png, .webp, .gif
